# ①課題名
ブックマークアプリ

## ②課題内容（どんな作品か）
- PHPとMySQLを用いて作成した、ログイン機能付きのブックマーク管理アプリ。
- ユーザーはログイン後、URL・タイトル・コメントを登録／一覧表示／編集／削除できる。
- 加えて、ユーザーは書籍名・URL・コメントを検索できる。
